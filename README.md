## Project Explanation

This Digital Clock project is developed using HTML, CSS, and Vanilla JavaScript.
The project displays the current time in hours, minutes, and seconds.
JavaScript Date() object is used to fetch real-time system time, and setInterval()
is used to update the clock every second automatically.
CSS is used to design a clean and modern user interface, while HTML provides
the basic structure of the clock.
This project helps beginners understand DOM manipulation, timing functions,
and real-time data handling in JavaScript.
